# gulpPack
## Клонируем и удаляем папку git:
```
git clone https://github.com/myfreelanceblog/gulpPack.git .
rd /s /q .git
```
## Инициализация проекта:
```
npm init`
```
## Установка версии webp:
```
npm install webp-converter@2.2.3 --save-dev
```
## Запуск версии разработчика:
```
npm run dev
```
